## Summary

This flow diagram illustrates how the dynamic membership pricing system works from end to end:

1. **Admin initiates** price change through UI
2. **Frontend validates** and sends API request
3. **Backend authenticates** and validates request
4. **Database updates** membership price
5. **Calculations auto-run** for derived values
6. **System-wide impact** reflects new prices everywhere

The entire process is secure, validated, and immediate.
